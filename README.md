# 🤖 Agente Inteligente de Incidencias de Software

Agente que gestiona bugs usando IA (Claude API) y permite crear, listar y resolver incidencias por consola y web.

## 🚀 Funcionalidades
- Crear incidencias
- Listar incidencias
- Resolver incidencias
- Ver estadísticas

## ▶️ Ejecutar
```bash
pip install -r requirements.txt
python main.py
```

## 🌐 Web
```bash
python app.py
```
